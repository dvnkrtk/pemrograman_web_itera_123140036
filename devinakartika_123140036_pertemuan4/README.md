# Program Pengelolaan Data Nilai Mahasiswa  

---

## Deskripsi Program
Program ini dibuat untuk **mengelola data nilai mahasiswa** menggunakan bahasa pemrograman **Python**. Tujuannya adalah untuk melatih pemahaman dasar Python dalam pemrograman backend — mencakup struktur data, fungsi, perhitungan, dan input/output.

---

## Fitur Program
✅ Menyimpan data mahasiswa dalam **list of dictionary**  
✅ Menghitung **nilai akhir** berdasarkan bobot (UTS 30%, UAS 40%, Tugas 30%)  
✅ Menentukan **grade (A–E)** berdasarkan nilai akhir  
✅ Menampilkan data dalam bentuk **tabel rapi di terminal**  
✅ Menambahkan data mahasiswa baru secara dinamis  
✅ Mencari **mahasiswa dengan nilai tertinggi dan terendah**  
✅ **Filter mahasiswa** berdasarkan grade  
✅ Menghitung **rata-rata nilai akhir kelas**
